# 데이터 임포트 스크립트

## 통계데이터포털 CSV 파일 임포트

### 사용 방법

1. CSV 파일을 `data/` 폴더에 저장
2. 다음 명령어 실행:

```bash
tsx scripts/import-medicines-from-csv.ts data/medicines.csv
```

### CSV 파일 형식

다음 컬럼을 인식합니다:

- **품목명** 또는 **제품명**: 약 이름
- **주성분** 또는 **성분명**: 주요 성분
- **업체명** 또는 **제조사**: 제조 회사
- **분류명** 또는 **약효분류**: 약물 분류
- **효능효과** 또는 **효능**: 효능
- **용법용량** 또는 **용법**: 복용 방법
- **사용상주의사항** 또는 **주의사항**: 주의사항
- **부작용**: 부작용

### 자동 분석 기능

스크립트는 약물명, 성분, 효능, 주의사항을 분석하여 자동으로 다음 특성을 판단합니다:

- **졸음 유발**: 항히스타민제, 수면제 등
- **각성 효과**: 카페인, 갑상선호르몬 등
- **위장 자극**: NSAID, 항생제 등
- **식사 시간**: 식전/식후/식사 중

### 예제

```bash
# 통계데이터포털 CSV 파일 임포트
tsx scripts/import-medicines-from-csv.ts data/statistics-portal-medicines.csv

# 결과

#   진행 중... 50개 저장됨
#   진행 중... 100개 저장됨
#   ...
# ✅ 완료!
#   - 성공: 850개
#   - 스킵: 150개 (중복)
#   - 전체: 1000개
```
